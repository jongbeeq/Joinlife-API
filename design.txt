REGISTER
POST /auth/register
BODY firstName,lastName,emailOrMobile,username,password,birthDate,gender
RESPONSE
201 { accessToken: String} 
400 { message: String}
500 { message: String}

LOGIN
POST /auth/login
BODY username, password
RESPONSE
200 { accessToken: String, user: { id: Number, firstName: String ....} }
400 { message: String }
500 { message: String }
